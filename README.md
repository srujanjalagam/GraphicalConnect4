**Instructions to play the game:** 
-	Create a folder called “Connect 4”
-	Download all of the files into the folder
-	Unzip the zip files into the newly created “Connect 4” folder
-	Unzipping the files into the folder will create two folders called “Assets” and “Source Code”
-	Copy all of the contents from "Assets" and "Source Code" into another new folder, can be called anything you want
-	Make sure to have an application that can run Java files
-	Compile the Main file by running “javac Main.java”
-	Run the game by running “java Main.java”
-	Enjoy!
